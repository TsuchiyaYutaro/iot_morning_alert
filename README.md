# iot_morning_alert
m5stack を利用した IoT 目覚まし時計
- 朝のアラートの設定
- アラートを止めるにはデバイスを振らなければならない
- 朝起きた時間を slack 上に通知

# Device
M5Stack Gray（9軸IMU搭載）<br>
https://www.switch-science.com/catalog/3648/

# Environment
- Arduino IDE
- Wi-Fi 
- slack

# Setup
以下入力
- HTTPS の中間証明書
- Wi-Fi の SSID, PASSWORD
- slack の WebHooks の `services/~`
